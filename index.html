hx

Haskell, at lightspeed

A blazingly fast package manager and build tool for Haskell, written in Rust. One tool to replace them all.

$curl -fsSL https://get.raskell.io/hx | sh

Or install via GitHub Releases or other methods

Get started GitHub

Why hx?

Everything you need to build Haskell projects, unified in a single tool that's 10-100x faster than existing solutions.

Instant resolution

Dependency resolution in milliseconds, not minutes. hx uses a modern SAT solver with aggressive caching to make waiting a thing of the past.

Unified toolchain

One tool for everything: create projects, manage dependencies, build, test, run, and publish. No more juggling between cabal, stack, and ghcup.

Drop-in compatible

Works with your existing .cabal files and Hackage packages. Migrate in seconds, not hours. Your workflow stays the same, just faster.

GHC management

Automatic GHC version management. hx downloads, installs, and switches compiler versions seamlessly based on your project requirements.

Reproducible builds

Lock files by default. Every build is reproducible across machines and CI. Share your lockfile, share your exact build.

Written in Rust

No runtime dependencies. Single static binary. Native performance with memory safety. Install once, run anywhere.

Why hx?

Haskell’s existing build tools are powerful but slow. hx brings Rust-level performance to Haskell package management.

Features

  • Fast dependency resolution — Parallel downloads and caching
  • Drop-in replacement — Works with existing .cabal files
  • Project management — Create, build, and manage Haskell projects
  • Modern CLI — Clean interface with helpful error messages

Quick Start

# Install hx
curl -fsSL https://get.raskell.io/hx | sh

# Create a new project
hx new my-project
cd my-project

# Build and run
hx build
hx run
